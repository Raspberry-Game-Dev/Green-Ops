Green Ops

A downloadable game for Windows and Linux

Will be available in 3 days.

Green Ops is based off old school classic shooter games where you have to fight your way from A to B.

Play as a Green Ops member where nothing can stand in your way. Shoot your rifle and throw your grenades to defeat the Red Hats.

The Red Hats are highly trained to take out green ops members so be warned.

Battle through seven different levels more levels coming soon! 


Player Controls

Move Left:    A key

Mover Right:   D key

Jump:    W key

Shoot:    Spacebar

Throw Grenades:   Q key

Options Menu:  O key


Install Instructions

Window users:

Unzip file and run GreenOps Executable

Linux users:

Unzip file and run Green Ops Executable.

You may need to install some additional packages for different distros  

works out the box on Pop!_OS 20.04 LTS x86_64
